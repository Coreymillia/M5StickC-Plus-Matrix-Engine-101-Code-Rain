# Matrix101 M5StickC Plus - 101 Matrix Rain Effects

**Version 1.0.1** - The ultimate Matrix rain experience for M5StickC Plus devices!

## Features
- **101 unique matrix rain effects** - From classic green rain to advanced physics simulations
- **Auto-advance modes** - Automatically cycles through effects every 30 seconds (configurable)
- **Simple controls** - Power/B buttons navigate, A button controls timing
- **Full screen effects** - Optimized for M5StickC Plus 135x240 display
- **Universal compatibility** - Works on both M5StickC Plus and Plus2

## Controls
- **Power Button**: Previous effect
- **Button B**: Next effect  
- **Button A**: Toggle auto-advance timing (30s → 1m → 10m → 1h → off)

## Installation
1. Open M5Burner
2. Load `Matrix101_M5StickC-v1.0-MERGED.bin`
3. Flash to your M5StickC Plus/Plus2
4. Enjoy 101 different Matrix experiences!

## Technical Details
- **Flash Size**: 4MB
- **Display**: 135x240 optimized
- **Memory**: Efficient column-based rendering
